# SAIDGILA — Pengurus Nota & Kewangan Futuristik 🇲🇾
> 🚀 Futuristic offline web app for work, study, activity, and finance notes with OT calculator & Malaysian calendar (2026–2050). Bilingual BM/EN. Built with vanilla HTML/CSS/JS. Ready for Netlify.
Aplikasi web tanpa server untuk mengurus nota harian, kewangan, dan pengiraan gaji overtime. Direka dengan tema neon futuristik, dwibahasa (Melayu & Inggeris), dan menyokong kalendar Malaysia dari Mei 2026 hingga Mei 2050.
## ✨ Ciri-ciri Utama
- 📅 **Kalendar Malaysia** lengkap dengan cuti umum, hujung minggu (Sabtu & Ahad), dan tarikh istimewa (23 Oktober — Hari Lahir Pencipta).
- 💼 **Nota Kerja** – tambah nota harian, alarm peringatan, dan kalkulator OT (1.5x/2.0x/3.0x mengikut jenis hari).
- 📚 **Nota Belajar** – catat nota pembelajaran dengan alarm.
- 🏃 **Nota Aktiviti** – rekod aktiviti harian.
- 💰 **Nota Finance** – masukkan pendapatan part-time, duit masuk/keluar, serta ringkasan kewangan bulanan.
- ⚙️ **Setting** – tetapan gaji basic, potongan KWSP, elaun, hari bekerja, jam bekerja. Paparan pendapatan bersih bulanan termasuk OT dan part-time.
- 📤 **Export & Kongsi** – cetak sebagai PDF (Ctrl+P), kongsi ke WhatsApp, Telegram, Email, atau salin ke clipboard.
- 🌐 **Dwibahasa (BM/EN)** – tukar bahasa dengan satu klik.
- 💾 **Simpanan Lokal** – semua data disimpan dalam `localStorage` pelayar; tiada pangkalan data luaran diperlukan.
## 🛠 Teknologi
- **HTML5** (struktur & semantic elements)
- **CSS3** (custom properties, flexbox, grid, animasi)
- **JavaScript ES6+** (DOM manipulation, localStorage, Web APIs)
- **Zero dependencies** – 100% vanilla, berfungsi sepenuhnya secara *offline* selepas halaman dimuat.
